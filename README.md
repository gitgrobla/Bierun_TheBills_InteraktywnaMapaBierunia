# Interaktywna mapa Bierunia

## Installation

```powershell
cd ./bierunapp
npm i
npm run dev
```

```powershell
cd ./bierunbackend
yarn
yarn run dev
```
## Requirements
In order to run the application and experience its fullest potential You need to have access to .fbx files. We do not keep them in our repo because of their sheer size.
